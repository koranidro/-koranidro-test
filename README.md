# @koranidro/test

## 1. 설치

```bash
npm install @koranidro/test;
```

## 2. 사용법

```javascript
import { add } from "@koranidro/test";

console.log(`sum = ${add(5, 1)}`);
```

또는

```javascript
const { add } = require("@koranidro/test");

console.log(`sum = ${add(5, 1)}`);
```

## 3. 실행 결과

```bash
sum = 6
```
